# Project Brief: KYA (Know Your Agent) — Solana Backend

## Vision

Платформа KYA связывает **понимание намерения агента/пользователя** (через LLM) с **неизменяемой фиксацией решений** в блокчейне Solana, чтобы решения были проверяемыми и привязанными к прозрачному пайплайну.

## Scope (текущий эпик)

1. **Backend на двух рантаймах:** Node.js и FastAPI — совместно обслуживают API и интеграции (точное разделение — в PLAN).
2. **Google Gemini API:** извлечение и нормализация интентов (controlled JSON: decision, reasoning, risk_level).
3. **Solana (Anchor):** запись итогового «решения» (структура полей — в PLAN) в on-chain программу.
4. **MCP-сервер (день 3):** доступ к возможностям KYA из AI-инструментов (Cursor и др.).

## Success criteria (черновик)

- Запрос к API проходит путь: анализ интента → принятие политики/валидация → запись в Solana → возврат `signature` и человекочитаемого результата.
- Devnet/mainnet и секреты описаны в конфигурации без утечек в репозиторий.
- MCP-сервер запускается локально и документирован.

## Out of scope (пока)

- Публичный фронтенд.
- Полный KYC/юридический compliance (если понадобится — повышает сложность до Level 4).

## Stakeholders

Команда KYA; backend-разработчик (вы) — владелец интеграций Gemini + Solana + MCP.
