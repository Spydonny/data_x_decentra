# Product

## What is KYA?

KYA brings a "Know Your Agent" trust primitive to Solana. It gives agent builders a way to register autonomous actors, define operating boundaries, and keep a verifiable history of decisions.

## Target Users

- Agent builders and runtime teams
- Treasury and wallet operators
- Protocols that need policy-gated automation
- Observers, auditors, and regulators

## Core Value Propositions

1. **Verifiable identity** - agents are represented through on-chain records instead of opaque wallet addresses.
2. **Policy-aware execution** - intent review can include mission context, limits, and human-readable reasoning.
3. **Auditable behavior** - decisions can be logged to Solana and queried later.
4. **Flexible integration** - the same trust layer is available through REST, MCP, and a browser dashboard.

## Why It Matters

Without KYA, an autonomous agent is just a wallet with hidden logic. With KYA, that wallet becomes a trackable actor with policy signals, decision history, and a public reputation surface.

## Delivery Surfaces

- REST API for apps and dashboards
- MCP tools for agent frameworks
- React dashboard for human operators
- Anchor program for on-chain state
